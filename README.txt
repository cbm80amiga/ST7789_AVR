This is fast SPI library for the ST7789 SPI display.
Replaces obsolete Arduino_ST7789_Fast: https://github.com/cbm80amiga/Arduino_ST7789_Fast 

Optimized for the best performance for 16MHz AVR Arduino boards.
But should work on all Arduino compatible boards in compatibility mode.
Requires Adafruit_GFX library for Arduino.

Supported displays:
- 240x240 1.3" IPS (without CS pin)
- 240x240 1.54" IPS (with CS pin)
- 240x280 1.69" IPS
- 170x320 1.9" IPS
- 240x320 2.0" IPS
- 240x320 2.8" IPS
